# Java Learning Materials

This folder contains comprehensive Java learning materials covering fundamental to advanced concepts.

## Contents

### Basic (Basic/)
- **ArrayExamples.java** - Array concepts, declaration, initialization, and manipulation
- **ControlFlow.java** - If-else, switch statements, conditional logic
- **DataTypes.java** - Primitive types, wrapper classes, type casting
- **HowJavaWorks.java** - JVM, compilation, execution, bytecode concepts
- **Loops.java** - For, while, do-while loops, loop control statements
- **Methods.java** - Method declaration, parameters, return types, overloading
- **Strings.java** - String class, string manipulation, immutability
- **VariablesAndOperators.java** - Variables, operators, operator precedence

### OOP (OOP/)
- **Abstraction.java** - Abstract classes, abstract methods, interfaces
- **AccessModifiersAndKeywords.java** - Access modifiers, static, final, this, super and other keywords
- **ClassesAndObjects.java** - Class structure, objects, constructors, instance variables
- **Encapsulation.java** - Data hiding, getter/setter methods, access control
- **Inheritance.java** - Single inheritance, method overriding, super keyword
- **Interfaces.java** - Interface declaration, implementation, multiple inheritance
- **Polymorphism.java** - Method overloading, method overriding, runtime polymorphism
- **StaticKeyword.java** - Static variables, static methods, static blocks

### Advanced (Advanced/)
- **CollectionExamples.java** - List, Set, Map, ArrayList, HashMap, TreeSet
- **Generics.java** - Generic classes, generic methods, wildcards, type parameters
- **ExceptionHandling.java** - Try-catch, finally, throws, custom exceptions
- **Multithreading.java** - Thread creation, synchronization, concurrent programming
- **StreamsAndLambda.java** - Functional interfaces, lambda expressions, streams API

### Interview Preparation (InterviewPreparation/)
- **ArrayProblems.java** - Common array manipulation problems
- **DataStructures.java** - Implementation and usage of data structures
- **NumberProblems.java** - Mathematical and number-based problems
- **StringProblems.java** - String manipulation and pattern problems
- **DynamicProgramming.java** - DP concepts and optimization techniques

## How to Use

1. Each file is self-contained with comprehensive documentation
2. Read the comments and examples to understand concepts
3. Modify and experiment with the code
4. Use as reference material for interviews and learning

## Structure

Each Java file follows this pattern:
```
- Package declaration
- Import statements
- Class documentation (Javadoc)
- Concept explanation with examples
- Helper methods and test cases
- Main method (where applicable) for testing
```

## Learning Path

1. Start with **Basic** folder to understand fundamentals
2. Move to **OOP** folder to master object-oriented concepts
3. Explore **Advanced** folder for complex topics
4. Practice with **InterviewPreparation** folder for problem-solving
